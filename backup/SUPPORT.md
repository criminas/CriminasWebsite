# Support

Getting help with arcbase projects.

## How to Get Help

### Documentation
Check the project README and any `docs/` or `DEVELOPMENT.md` files in the repository.

### Community Discussion
For general questions and feature discussions, open an issue on GitHub.

### Reporting Bugs
Please use the [Bug Report](https://github.com/arcbasehq/.github/blob/main/ISSUE_TEMPLATE/bug_report.yml) template.

### Security Issues

> **Do NOT report security vulnerabilities via public issues.**

Email security concerns directly to: **arcbase@tuta.io**

We appreciate your responsible disclosure and will work with you to address the issue.

## Contributing

Interested in contributing? Check out our [CONTRIBUTING.md](https://github.com/arcbasehq/.github/blob/main/CONTRIBUTING.md) and look for `good first issue` labels.
