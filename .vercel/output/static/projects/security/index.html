What is ArcOSINT?

ArcOSINT is a comprehensive, terminal-based OSINT framework built for ethical security research. It brings powerful investigation modules together with a clean, intuitive TUI for fast, focused analysis.

From email and username intelligence to IP, domain, phone, and metadata investigations, ArcOSINT keeps everything local, privacy-aware, and easy to export.
